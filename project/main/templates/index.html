Что это за сайт?

Vortex - это простой и интуитивно понятный онлайн-редактор данных, который позволяет пользователям легко редактировать и модифицировать различные типы файлов, включая текст, код Python, JavaScript, C++ и многое другое.

Основные функции и возможности сайта:

  1. Поддержка различных форматов файлов:

    Vortex поддерживает широкий спектр форматов файлов, включая txt, py, js, и cpp, а также другие популярные языки программирования.

  2. Удобный интерфейс:

    Интерфейс сайта разработан с учетом простоты и удобства использования. Пользователи могут выбирать файлы для редактирования, просматривать их содержимое и вносить изменения, используя различные инструменты форматирования и редактирования.

Также, вы сами можете сделать локальную версию сайта

Инструкция:
  1. Скачиваем саму программу (https://gitlab.informatics.ru/2023-2024/korolev/s101/vortex_py)

  2. Устанавливаем python 3.10

  3. Устанавливаем все необходимые библиотеки:
    • Django 5.0.2 (pip3 install django==5.0.2)

  4. Из папки проекта (vortex_py) запускаем команду python3 project/manage.py runserver ID:PORT. Вместо IP - ip адрес, а вместо PORT - порт, с которых будет запущен сервер.
    Если Вы хотите просто запустить в локальной сети, то python3 project/manage.py runserver

  5. Всё готово, переходите по своему адресу (или localhost:8000)
